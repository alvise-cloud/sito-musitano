# Versione statica senza pagamento

Questa cartella contiene la versione pubblicabile gratis tramite GitHub Pages o caricabile su SiteGround come sito statico.

Limiti della versione statica:

- non usa Flask in produzione;
- non ha database prenotazioni;
- non ha area admin funzionante;
- il modulo prenotazione apre email e WhatsApp invece di salvare dati nel pannello admin.

Questa versione e stata preparata per evitare piattaforme che richiedono carta o pagamento.
